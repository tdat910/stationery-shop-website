# Stationery shop 
A stationery store management system built with Laravel following the MVC (Model-View-Controller) pattern. This project provides an efficient solution for managing office supplies, and basic sales operations.

# Introduction
Stationery Store is a web application designed to help manage and sell office supplies such as pens, notebooks, and other materials. The project focuses on applying MVC architecture, database management, and basic e-commerce features.

# Key Future
* **Authentication & Authorization:** Register, login via Gmail, logout, and role-based access control system.
* **Product Management:** View products, details, and admin performs full CRUD operations easily.
* **Shopping Cart Management:** Add, update, remove products, and manage cart items efficiently.
* **Order Management:** Place orders, enter address, choose payment, and admin manages orders.
* **Order History & Cancellation:** Users view order history and cancel orders when needed.
* **Search & Product Discovery:** Search, filter, and browse products by categories for better experience.
* **AI Recommendation System:** Suggest products based on user behavior and purchase history data.
* **Database Management:** Manage relational database for users, products, carts, and orders system.
* **System Administration & Security:** Handle authentication, sessions, and secure role-based system access.

# Technologies Used
*   Language: PHP (MVC Pattern).
*   Database: MySQL (Using PDO & Singleton Pattern).
*   Frontend: HTML, CSS, JS.
*   Server: Apache (XAMPP).
